# ChatBoy AI

A simple AI chat app using OpenAI API and Cloudflare Workers. Frontend works on any browser, backend runs securely on Cloudflare.

## Features
- Chat with an AI powered by OpenAI
- Frontend works on mobile and desktop
- Secure API key storage in Cloudflare Worker
- CORS ready
